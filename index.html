Игра не запускается

После применения любого исправления проверьте работу игры перед переходом к следующему шагу.

Проверка пути установки

  • Установите клиент на диск с Windows (обычно C:)
  • Рекомендуемый путь: C:\Games\PVP_Moldova
  • Избегайте путей с кириллицей

Метод запуска

  • Запустите скрипты разрешения и языка из папки Scripts
  • Всегда запускайте через Launcher или main.exe
  1. Правый клик → Свойства → Совместимость
  2. Выберите Windows 7
  3. Отметьте "Запуск от имени администратора"
  4. Применить

.NET Framework 3.5

Сетевые проблемы

Выполните:

netsh winsock reset

DEP

Добавьте main.exe в исключения DEP и перезагрузите ПК.

Вылеты и зависания

Некоторые антивирусы требуют полного удаления.
  • Добавьте игру в исключения
  • Проверьте Windows Defender
  • Отключите UAC
  • Проверьте DEP
  • Просканируйте систему через Malwarebytes
  • Настройте видеокарту (NVIDIA/AMD)

Лаги / Низкий FPS

Лучший FPS иногда достигается при запуске напрямую через main.exe.
  • Обновите драйверы
  • Отключите HD
  • Min Colors
  • Отключите 3D Camera
  • Settings.ini → New FPS = 1
