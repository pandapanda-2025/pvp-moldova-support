PVP Moldova – Руководство по устранению неполадок

Игра не запускается

После применения каждого решения проверьте запуск игры перед тем как переходить к следующему шагу.

Путь установки

  • Установите клиент на диск с Windows (обычно C:)
  • Рекомендуемый путь: C:\Games\PVP_Moldova
  • Не используйте кириллицу в названии папок

Правильный запуск

  • Запустите скрипты разрешения/языка из папки Scripts
  • Всегда запускайте через Launcher или main.exe
  • Запуск от имени администратора (режим совместимости Windows 7)

.NET Framework

Убедитесь, что установлен .NET Framework 3.5.

DirectX

  • Установите DirectX 9.0c или новее
  • Проверьте версию через dxdiag (Win + R)

Сброс сети

Откройте командную строку и выполните:

netsh winsock reset

После этого перезагрузите компьютер.

DEP (Data Execution Prevention)

  • Этот компьютер → Свойства → Дополнительные параметры системы
  • Быстродействие → Параметры → Предотвращение выполнения данных
  • Добавьте main.exe в исключения

Игра вылетает или зависает

Антивирус блокирует игру

Некоторые антивирусы требуют полного удаления, обычного отключения может быть недостаточно.
  • Добавьте папку PVP Moldova в исключения
  • Проверьте Windows Defender
  • Временно отключите защитное ПО

Ошибка прав администратора

Если появляется красное предупреждение — отключите UAC (Контроль учетных записей).

Проблемы с видеокартой

  • Принудительно выберите видеокарту в панели NVIDIA / AMD
  • Протестируйте интегрированную и дискретную графику

Низкий FPS / Лаги

Некоторые игроки получают лучший FPS, обновляя через Launcher, но запуская игру напрямую через main.exe.

Базовые шаги

  • Обновите драйвер видеокарты
  • Понизьте графику в игре
  • Закройте фоновые программы
  • Проверьте стабильность интернета

Продвинутые настройки

  • Понизьте разрешение
  • Отключите HD в Launcher
  • Включите Min Colors
  • Отключите 3D Camera
  • В файле Settings.ini установите New FPS = 1
