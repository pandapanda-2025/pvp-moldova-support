Если игра не запускается

Типичные проблемы при запуске

1. Проверка пути установки

  • Убедитесь, что клиент установлен на тот же диск, где находится Windows (обычно диск C:).
  • Рекомендуемый путь: C:\Games\PvP Moldova
  • Избегайте путей с кириллицей (пример: C:\Игры\PvP Moldova).

2. Способ запуска игры

  • Запустите скрипты разрешения и языка из папки Scripts.
  • Всегда запускайте через Launcher или main.exe.
  • Запуск от имени администратора:
  1. Правая кнопка мыши main.exe → Properties → Compatibility
  2. Отметить Windows 7
  3. Отметить "Запустить программу от имени администратора"
  4. Применить

3. Распаковка файлов игры

Убедитесь, что все файлы игры полностью извлечены.

Технические требования

1. .NET Framework 3.5

Убедитесь, что установлен корректно.

2. DirectX

  • Убедитесь, что установлен DirectX 9.0c или новее.
  • Обновите DirectX до последней версии.
  • Проверка версии: Win+R → dxdiag

3. Сетевые проблемы

  • Запустите скрипты из папки Scripts.
  • Откройте командную строку и выполните: netsh winsock reset
  • Перезагрузите ПК.
  • При необходимости смените IP или перезапустите интернет.

Предотвращение выполнения данных (DEP)

Если появляется ошибка «main.exe has stopped working»:

  1. Щёлкните правой кнопкой мыши по «Этот компьютер» → выберите «Свойства»
  2. Нажмите «Дополнительные параметры системы»
  3. Во вкладке «Дополнительно» в разделе «Быстродействие» нажмите «Параметры»
  4. Перейдите во вкладку «Предотвращение выполнения данных»
  5. Выберите подходящий вариант и добавьте main.exe в исключения
  6. Нажмите «Применить» и перезагрузите компьютер

Игра вылетает или зависает

Конфликты с защитным программным обеспечением

ВНИМАНИЕ: Некоторые антивирусные программы требуют полного удаления.
  • Проверьте, блокирует ли антивирус игру.
  • Добавьте папку MU в исключения.
  • Windows Defender
  • Norton / Bitdefender / RAV (может потребоваться удаление)
  • Avira, AVG и другие

Windows Security Features

  • Если появляется ошибка прав администратора — отключите UAC.
  • this tutorial

System Cleanliness

Проверьте систему через: Malwarebytes Anti-Rootkit

Graphics Driver Issues

Dual Graphics Card Setup

  • Откройте NVIDIA Control Panel или AMD Radeon Settings
  • Найдите PvP Moldova (main.exe)
  • Назначьте дискретную или интегрированную видеокарту
  • Протестируйте лучший вариант

Проблемы с производительностью

Лаги и низкая производительность

Некоторые игроки получают более высокий FPS, если используют Launcher только для обновлений и запускают игру напрямую через main.exe.

1. Основные действия

  • Обновите драйверы
  • Понизьте графику
  • Закройте фоновые программы
  • Проверьте интернет
  • Проверьте минимальные требования

2. Дополнительные настройки производительности

  • Понизьте разрешение
  • Отключите HD
  • Min Colors
  • Effect → Antilag
  • Отключите 3D Camera
  • Settings.ini → New FPS = 1
